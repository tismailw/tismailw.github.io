Keylogger

Description:


    This project was created out of my passion and desire to learn zybersecurity basics. In an attempt to learn how keyloggers work, I started to research and learn how keyloggers work. Doing this allowed me to learn how a keylogger works, how to avoid being a victim to one, and how I can enhance a project. Initially this project only had the function to store keys and output them thrpugh a IDE. I then had to imagine how would someone use this in the real world, using this, I added another functionality of sending it to an email.

    Note: Creating this small project was out of my passion for cybersecurity. It is not advisable to use it for malicious use.
