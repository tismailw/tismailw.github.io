Medical Assistance Accessibility Tool - Made for John Hopkins Hackathon

Description:


    "The Medical Assistance Accessibility Tool" is built using these tools: chatGPT API, HTML, CSS, JavaScript, WebSpeechKit library, and Google Cloud. It can listen to extensive medical conversations and by using Ai infrastructure and trained data models, it can translate the complex medical information into a much simpler version of it. Additionally, this tool can simulate a consultation with physicians. Both of these functions are seamlessly integrated into a friendly user interface.


    The purpose of this project is to assist marginalized communities like the blind, illiterate, and dissabled, in overcoming communication barriers in the medical industry. Our teams efforts, which I spearheaded, paid off. In the 36 hours of working on this project, we were able to place in the top 3 of the BGB Group track.
